El software "Proyecto Amigo Secreto" es una aplicación web que permite a los usuarios ingresar nombres de amigos y realizar un sorteo aleatorio para determinar el "amigo secreto". Los usuarios pueden agregar nombres mediante un campo de texto, ver la lista de amigos, y realizar el sorteo con un solo clic.
